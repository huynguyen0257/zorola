# Daily Action Log: 2026-04-15

## Context

Today we defined the product direction and built the first prototype for collecting images from Zalo Web. The user wants full automation eventually, but the immediate proof point is: can we pull images from a Zalo group into local files?

## Decisions Made

- Use Zalo Web instead of Zalo Desktop.
- Start with macOS command-line usage.
- Later add a `.command` double-click runner for another macOS machine.
- Do not ask publishers to change where they post images.
- Treat each publisher submission as a cluster of images.
- Treat Google Sheets as an audit log in Zalo display order.
- Keep uncertain account extraction results reviewable instead of guessing.
